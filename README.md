# ALX Listing App

## Description 
A modern Airbnb built with Next.js, TypeScript, and TailwindCSS. This project demonstrates best practices in building a scalable and maintainable web application with a focus on property listings.

---

## Project Structure

The project follows a clean and organized structure:

- `components/`: Reusable UI components
  - `common/`: Basic components used throughout the app
    - `Button.tsx`: Customizable button component
    - `Card.tsx`: Property listing card component
- `interfaces/`: TypeScript interfaces and types
- `constants/`: Application-wide constants and configuration
- `public/assets/`: Static assets (images, icons)
- `pages/`: Next.js pages and API routes
- `styles/`: Global styles and Tailwind CSS configuration

---

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Dayvid0063/alx-listing-app.git
```

2. Install dependencies:
```bash
cd alx-listing-app
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
